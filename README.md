## Featured Project: AI Knowledge Assistant

A corporate-style internal assistant that performs fast semantic search inside Slack.  
It replaces bulky documents with instant answers.

### Highlights
- Local vector search using **FAISS**
- MiniLM embeddings for high relevance
- Structured, filtered answers (no hallucination)
- Contact enrichment via contacts.csv
- Runs offline, no paid LLM required
- Comes with a **Sandbox**:
  - Batch question testing  
  - Automated evaluation  
  - Retraining pipeline  
  - Logs + analytics
